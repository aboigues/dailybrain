# Fonts

This directory should contain the following font files as specified in the design system (section 4.1):

## Poppins (Headings)
- `Poppins-Regular.ttf`
- `Poppins-SemiBold.ttf` (weight: 600)
- `Poppins-Bold.ttf` (weight: 700)

## Inter (Body)
- `Inter-Regular.ttf`
- `Inter-Medium.ttf` (weight: 500)

## JetBrains Mono (Scores)
- `JetBrainsMono-Regular.ttf`

## Download Links

- **Poppins**: https://fonts.google.com/specimen/Poppins
- **Inter**: https://fonts.google.com/specimen/Inter
- **JetBrains Mono**: https://fonts.google.com/specimen/JetBrains+Mono

## Installation

1. Download the font files from Google Fonts
2. Place them in this directory
3. Ensure filenames match those specified in `pubspec.yaml`
