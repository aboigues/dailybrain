# Images

This directory will contain image assets for the DailyBrain app.

## Required Images

### Logo
- `logo.png` - Main app logo (1024x1024 for app stores)
- `logo_small.png` - Small logo for splash screen (512x512)

### Onboarding
- `onboarding_brain.png` - Brain illustration for welcome screen

### UI Elements
- `empty_state_quiz.png` - Empty state when no quiz available
- `empty_state_ranking.png` - Empty state for rankings

### Streak Badges
- `fire_emoji.png` - Streak icon (if not using system emoji)
- `trophy_*.png` - Trophy icons for achievements

## Format Guidelines

- **Format**: WebP preferred (smaller size), PNG fallback
- **Resolutions**: Provide 1x, 2x, 3x versions for different screen densities
- **Optimization**: All images should be optimized (use tools like TinyPNG)
- **Naming**: Use snake_case for file names

Per Constitution Principle I: App size must be < 30MB.
Keep image sizes minimal while maintaining quality.
